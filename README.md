# 📂 Como acessar o projeto completo

## 1. Clonar o repositório

Caso você tenha o Git instalado, execute:

```bash
git clone https://github.com/DevCage27/AboutMe.git
```

Depois entre na pasta do projeto:

```bash
cd AboutMe
```

Abra o projeto no Visual Studio Code:

```bash
code .
```

---

## 2. Baixar pelo GitHub

Também é possível baixar o projeto diretamente:

1. Acesse:

https://github.com/DevCage27/AboutMe

2. Clique no botão:

```
Code → Download ZIP
```

3. Extraia o arquivo baixado.

4. Abra a pasta no Visual Studio Code.

---

# 🖥️ Executando o projeto

Após abrir o projeto:

1. Localize o arquivo:

```
index.html
```

2. Abra o arquivo no navegador.

Ou utilize a extensão **Live Server** no VS Code:

1. Abra as extensões do VS Code.
2. Procure por:

```
Live Server
```

3. Instale a extensão.
4. Clique com o botão direito no arquivo `index.html`.
5. Selecione:

```
Open with Live Server
```
